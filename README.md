**Introduction**
This is the SpringBoot application, It includes the functionalities:
1. Spring-Boot MVC architecture (With thymeleaf template).
2. CRUD operations (with JDBC template & JPA).
3. Pagination.


**We have used the following technologies in this application:**
1. Technology: JAVA-8
2. Frameworks: SpringBoot, JPA
3. Database: MySQL
4. Build-Type: Maven
5. Profiles: UAT & PROD
